BLOCK ELEMENTS

Defination:

Block elements are those elements which starts from new line. And such type of elements take complent width of the screen.

Examples of block elements:

  1. p tag
  2. ol tag
  3. ul tag
  4. dl tag
  5. article tag
  6. section tag
  7. div tag

INLINE ELEMENTS

Defination:

Inline elements are those who starts from there where the previous element ends. Such type of elements take only required length taken by content of inline element.

Examples of inline elements:

  • anchor tag
  • span tag
  • b tag
  • em tag
  • mark tag
